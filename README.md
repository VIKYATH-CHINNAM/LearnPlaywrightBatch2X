# LearnPlaywright2X

A repository for learning Playwright and documenting JavaScript coding rules and conventions.

## Structure

- `learning/` - Playwright test examples and learning exercises
- `rules/` - Documentation for JavaScript coding standards and conventions
  - `comment_rules.md` - Commenting guidelines and VS Code shortcuts
  - `identifier-rules.md` - JavaScript identifier naming rules and reserved keywords
  - `shortcuts_window` - Windows keyboard shortcuts
  - `shorcuts_mac` - macOS keyboard shortcuts

## Getting Started

1. Install dependencies:
   ```bash
   npm init playwright@latest
   ```

2. Run tests:
   ```bash
   npx playwright test
   ```
